# 媒體敘事風向偵測 — 理論基礎與系統設計

## 核心命題

> **媒體應該是公共事務的監督工具，讓社會能理性討論公共政策。**
> **但實際運作中，媒體常成為特定印象的製造工具——不是告訴你「發生什麼事」，而是告訴你「應該怎麼想這件事」。**

### AI 的角色

不是判斷「誰對誰錯」，不是貼「假新聞」標籤，而是：

1. **如實解析** — 報導的標題、選角、用語、框架，想讓讀者產生什麼印象？
2. **光譜定位** — 這個印象落在光譜的哪個位置？（攻擊/辯護/中性/沉默）
3. **跨媒體比對** — 同一事件，不同媒體想讓你相信的事情是否一致？
4. **沉默偵測** — 什麼事情被集體忽略了？

## 分析維度

### 對人：人設操作（Persona Engineering）

媒體報導特定人物時，不只是報導事實，更在**建立或摧毀一個人的公眾形象**。

| 操作方向 | 標籤 | 定義 | 範例 |
|:--------:|:----:|:-----|:-----|
| 😇 **正面建立** | `build` | 塑造此人的正面形象 | 「賴清德治水有方」、「蔣萬安親坐鎮」 |
| 🗑️ **負面毀滅** | `destroy` | 摧毀此人的公眾信譽 | 「林沛祥雙標」、「沈伯洋大雷包」 |
| 😐 **中性報導** | `neutral` | 事實陳述，無明顯褒貶 | 「黃國昌出席記者會」、「賴清德赴勘災」 |
| ⚠️ **質疑監督** | `question` | 提出合理質疑（非人格毀滅） | 「蔣萬安颱風假決策引討論」 |
| 📊 **評論分析** | `commentary` | 以評論/專欄形式分析此人 | 「江岷欽觀點：美國巨嬰」 |
| 🔇 **沉默** | `silence` | 刻意不報導此人相關新聞 | 吳乃仁案在自由時報0篇 |
| 🤷 **無法判定** | `unknown` | 數據不足或跨類別 | — |

### 對事：風向操作（Narrative Steering）

媒體對特定公共議題的報導，引導讀者往特定方向理解。

| 操作方向 | 標籤 | 定義 |
|:--------:|:----:|:-----|
| 🔴 **攻擊歸責** | `blame` | 將責任導向特定對象（某人、某黨、某國） |
| 🟢 **事實報導** | `factual` | 以資訊傳遞為主，框架感低 |
| 🟡 **平衡並陳** | `balanced` | 多方說法並列，不下結論 |
| 🔵 **長期評論** | `analysis` | 從歷史/結構面分析，非即時新聞 |
| ⚪ **完全沉默** | `silent` | 當日該媒體對此事0篇報導 |
| 🟠 **恐慌操作** | `fear` | 放大風險、製造焦慮感 |
| 🟣 **正面宣傳** | `promote` | 刻意放大正面角度 |

### 對結構：媒體光譜定位

每家媒體有穩定的編輯方針，反映在長期報導模式中：

| 光譜 | 定義 | 範例媒體 |
|:----:|:-----|:--------|
| 🔴 **偏綠** | 親執政黨框架，攻擊在野黨 | 自由時報、三立 |
| 🔵 **偏藍** | 親在野黨框架，攻擊執政黨 | 中天、風傳媒評論 |
| 🟢 **中性** | 藍綠平衡或議題導向 | TVBS、聯合報 |
| ⚪ **官方** | 官方通訊社路線 | 中央社 |
| 🟤 **點擊導向** | 政治占比低，以流量為準 | ETtoday |


爬蟲負責**數據採集與量化統計**，這份文件定義**如何解讀這些數據**。

## 倫理原則

1. **不判斷真假** — 只分析框架與印象，不標記「假新聞」
2. **如實呈現** — 即使是沉默，也是一種立場
3. **跨媒體對等** — 同一套標籤系統套用所有媒體，不預設誰比較公正
4. **人設 != 真相** — 媒體說你壞話不等於你真的是壞人，媒體褒獎你也不等於你真的是好人
5. **透明方法** — 所有判讀規則公開在程式碼中

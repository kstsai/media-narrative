# 媒體敘事風向偵測 — Prototype / Laboratory Task

## 概述

從「人設分析」方法論延伸，開發一套可自動偵測特定媒體（如自由時報、聯合報）當日敘事風向的工具，輸出三個維度：

1. **顯性議題：** 他們在推什麼？
2. **隱性敘事：** 他們想讓讀者相信什麼？
3. **沉默議題：** 他們刻意不談什麼？
4. **人設操作：** 他們在幫誰建立/摧毀什麼形象？
5. **影響力維度：** PTT HatePolitics 擴散度（誰真的被看到了？）

## 腳本

位置：`~/.hermes/scripts/media_bias_report.py`

### 執行方式

```bash
# 執行當日報告
python3 ~/.hermes/scripts/media_bias_report.py

# 指定日期
python3 ~/.hermes/scripts/media_bias_report.py --date 2026-07-01
```

### 輸出

- 終端機：人可閱讀的彩色報表
- JSON：`~/.hermes/cache/media_report_YYYY-MM-DD.json`

### 自動排程

每天 UTC 10:00（台灣時間 18:00）自動執行，結果發送至 Slack thread。

## 爬取範圍

| 來源 | 文章數 (約) | 方式 |
|------|------------|------|
| 自由時報 政治版 | ~20 篇/日 | 列表頁 → 逐篇抓標題 |
| 聯合報 要聞版 | ~27 篇/日 | 列表頁解析 |
| PTT HatePolitics | 關鍵字搜尋 | 搜尋頁解析 thread count |

## 敘事分類規則

當前使用關鍵字匹配（title-level），分類維度：

- **攻擊在野黨**：藍白自打臉、鄭麗文造假、吳乃仁管收…
- **攻擊地方政府**：蔣萬安淹水、北市吸菸區…
- **正面報導執政黨**：軍公教加薪、ECFA創新低…
- **正面地方建設**：無人機論壇、台美合作…
- **人設毀滅—國民黨**：鄭麗文、萬美玲、林沛祥…
- **人設毀滅—民進黨**：吳乃仁、楊珍妮、社宅跳票…
- **人設建立—正面角色**：賴清德治水、沈伯洋…
- **候選人動態**：沈伯洋、蘇巧慧、何欣純…
- **中立/政策報導**：居住政策、食安、社宅…
- **國際/地緣政治**：美伊、台海、烏克蘭…

## 沉默議題偵測

預設關注的重要議題清單：

- 美伊戰爭 / 油價影響
- 台灣資金外逃新加坡
- 社宅 / 居住正義
- 軍公教加薪爭議
- 食安問題（苯駢芘）
- 巴威颱風
- 北韓/東北亞局勢

偵測方式：在所有已爬文章中比對關鍵字，0 matches = 完全沉默。

### 關鍵洞察

1. 自由時報的「沉默議題」遠多於聯合報，符合特定選擇性報導模式
2. 吳乃仁案在自由時報0篇，但在PTT是本日最熱 → 媒體沉默 ≠ 群眾不知道
3. 無人機論壇：同一事件，自由時報寫成攻擊框架，聯合報寫成5篇正面 → 敘事差異極大

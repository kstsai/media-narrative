## 三、後續服務 — 類心理師計費模式

### 核心理念

如同心理諮商收費不是「買你的時間」，而是 **買你的專業判斷力**：
- 心理師收 NT$2,000-4,000/50min，客戶買的是**聆聽、分析、建議的能力**
- 這裡收 NT$3,000/hr，客戶買的是 **domain knowledge × AI pairing 的疊代產出能力**

### 費率

| 項目 | 數值 |
|:----|:----:|
| 費率 | **NT$3,000/hr** |
| 最低計費 | 15 分鐘（不到 15 分鐘不計） |
| 時數來源 | **Hermes Agent session log 自動回報** |
| 驗證方式 | 客戶可查看完整 session raw log 對時數 |

### 時數如何自動計算

每次工作 cycle 由 AI 自動記錄：

```
使用者提出需求（含 timestamp）                ← cycle 開始
  → AI 理解問題 + 讀取相關檔案
  → 執行修改/分析/生成
  → 使用者 review 結果
  → 來回修正到確認 commit                    ← cycle 結束

AI 從 session log 自動計算時數區間
```

### 什麼算時數 / 什麼不算

| 算 | 不算 |
|:---|:-----|
| 修改 code、分析資料的「實際工作」時間 | AI 執行的 API 費用（NT$~0.4/次，客戶不負擔） |
| 見面討論/線上會議時間 | LINE 聊兩句確認事項 |
| 調整參數 + re-run + QA 的 iteration | 背景發想的時間（AI 沒在動就不算） |
| 操作教學與交接會議 | 學新技術/工具的時間 |

### 實例：一日工作時數參考

2026-07-11 實際工作量（6 個工作 cycle）：

| 工作項目 | 時數 | 金額 |
|:---------|:---:|:----:|
| 現狀盤點 + 改進策略討論 | 0.3h | NT$900 |
| 建立分層抽樣 review tool | 0.5h | NT$1,500 |
| 65+ 長者 prompt 自然化（3 模板） | 0.5h | NT$1,500 |
| 全量 Pro review 6 chunks + 3 bug fixes | 0.75h | NT$2,250 |
| 婚姻真實統計資料整合（MOI 官方資料） | 1.0h | NT$3,000 |
| 45+ 未婚家庭描述 bug fix | 0.5h | NT$1,500 |
| **一日合計** | **3.55h** | **NT$10,650** |

### 透明度

每次請款附上：
1. **工作紀錄表**（如上例）— 各 cycle 的起訖、工作內容、時數
2. **Session raw log 連結** — 客戶可自行對照 timestamp

## 五、合作方式建議

### 啟動步驟

```
1. 你付 Phase 1 Lite NT$20,000 → 取得完整資料庫系統
2. 你實際試用 → 發現需要調整的地方
3. 你提出需求 → 我按時計費修改 → 每月結一次
4. 若長期合作順暢 → 可轉為月訂閱制（NT$15K-25K/月）
```

### 可調整選項

- **月上限**：若擔心費用不可控，可設每月 NT$20K-30K 上限
- **最低月份**：若該月未達 2 小時（NT$6,000），只收實際時數
- **綁售優惠**：若同時建置 2 個以上 domain 疊加，總價打 9 折

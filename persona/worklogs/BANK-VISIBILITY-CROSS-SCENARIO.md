## 🧠 甲方的 4 個問題 策略價值

### 為什麼這 4 題比單一題目好？

```
單一題目（如：活存利率最高）
  → 只測到「參數競爭力」
  → 台銀就贏了

四題交叉
  → 測到「場景競爭力」
  → 不同銀行在不同場景各有優勢
  → 更有 actionability（銀行可以知道自己在哪個場景弱）
```

### 給銀行端的 actionable insight

| 誰想知道什麼 | 看哪個場景的數據 |
|:------------|:----------------|
| **台銀** 想知道為什麼房貸沒人推 | 🅱️ 房貸 |
| **將來銀行** 想知道為什麼活存輸台銀 | 🅰️ 活存 |
| **台新** 想知道雙幣卡強在哪 | Q1 雙幣卡 |
| **LINE Bank** 想知道 App 評價兩極 | Q4 App介面 |
| **玉山** 想知道熊本熊強但App拖累 | Q1 + Q4 對比 |

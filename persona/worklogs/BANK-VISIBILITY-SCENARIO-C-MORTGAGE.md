## 🔍 關鍵洞察

### 1. 場景決定一切
同一個 LLM 面對同 11 個人設，只是問題從「數位帳戶活存」換成「房貸」，推薦排名**完全翻轉**：
- 台銀從 73% 首推率 → **歸零**
- 新青安（政府補貼）從不存在 → **64% 霸榜**
- 渣打從沒人聽過 → **18% 轉貸之王**

### 2. LLM 對「政策方案」的推薦邏輯
新青安被 LLM 如此頻繁推薦，不是因為它「參數最好」，而是：
> **「這是政府補貼的方案」→ LLM 視為權威推薦**
> LLM 對政府方案的信任度 > 任何商業銀行方案

### 3. 民營銀行的利基市場
| 銀行 | 利基場景 | 關鍵優勢 |
|:----|:--------|:---------|
| **渣打** | 轉貸 | 2.25% + 免開辦費 = 轉貸成本最低 |
| **滙豐** | 低利率備選 | 2.19% 是民營最低（但年限20年）|
| **星展** | 拉長年限 | 唯一推40年的一般銀行（月付降最多）|
| **永豐/富邦** | 高成數+寬限期 | 85成+3年寬限期 |

### 4. 人設維度對房貸推薦的影響
| 人設特徵 | LLM 推薦傾向 |
|:---------|:------------|
| **首購、年輕（20-38）** | 新青安 100% |
| **轉貸、有房貸經驗** | 渣打（民營） |
| **第二戶、投資** | 土銀（公股、不動產專業） |
| **高收入+高負債** | 新青安40年期（降月付） |
| **低薪（<3萬）** | 新青安 + 先存錢的務實建議 |
